HematoVision

Transfer Learning Based Blood Cell Classification

Project Overview

HematoVision is an AI-based blood cell classification system developed using Transfer Learning and Deep Learning techniques. The system classifies blood cells into Eosinophil, Lymphocyte, Monocyte, and Neutrophil categories.

Technical Skills Used

  • Python
  • Flask
  • TensorFlow / Keras
  • Transfer Learning (MobileNetV2)
  • HTML, CSS, JavaScript
  • VS Code

How It Works

Step 1: The user uploads a microscopic blood cell image through the HematoVision web interface.

Step 2: The Flask web application receives the uploaded image and stores it temporarily on the server.

Step 3: The uploaded image undergoes preprocessing, including resizing and normalization, to match the input requirements of the trained model.

Step 4: The preprocessed image is passed to the trained Convolutional Neural Network (CNN) model developed using Transfer Learning.

Step 5: The model analyzes image features and classifies the blood cell into one of the categories such as Eosinophil, Lymphocyte, Monocyte, or Neutrophil.

Step 6: The prediction result along with confidence scores is sent back to the Flask application.

Step 7: Finally, the predicted blood cell type and result visualization are displayed to the user on the result page.

Applications / Uses

  • Medical diagnostics assistance
  • Laboratory automation
  • Medical education
  • Remote healthcare analysis
